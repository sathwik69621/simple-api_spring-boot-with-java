# Spring Boot Simple REST API

This project is a simple RESTful API developed using Java and Spring Boot.

## Endpoints
- GET /hello – returns a welcome message
- GET /info – returns basic JSON data

## Tools Used
- Java
- Spring Boot
- Postman

## How to Run
1. Clone the repository
2. Open in IDE
3. Run the Spring Boot application
4. Test using Postman
